# SnowRabbit（Unity Project）

## 説明(Description)

リポジトリルートに配置されたディレクトリ構造は Unity3D ゲームエンジン向け構造のため  
本体へのアクセスをする場合は、 [SnowRabbit（Packages/SnowRabbit/）](Packages/SnowRabbit/README.md) へアクセスしてください。
